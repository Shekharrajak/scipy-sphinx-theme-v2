NumPy
====


NumPy is het fundamentele pakket voor wetenschappelijke controle met Python.
Het bevat onder andere:

- een krachtig N-dimensional array object
- geavanceerde (radio) functies
- tools voor het integreren van C/C++ en Fortran code
- nuttige lineaire algebra, Fourier transformatie en willekeurige getalsmaterie
